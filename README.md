# 🚀 SmartGen AI

✨ A modern **AI SaaS landing page** built with pure HTML, CSS, and JavaScript — designed to simulate a real-world AI product experience with clean UI and smooth interactions.

---

## 🧭 Overview

SmartGen AI is a frontend-focused project that replicates the structure and feel of a modern AI-powered SaaS platform.
It demonstrates how real startup landing pages are designed with usability, responsiveness, and user engagement in mind.

---

## ✨ Highlights

⚡ Fast and responsive UI
🎯 Clean, minimal, and modern design
🌙 Dark / Light theme toggle
💬 Interactive AI response simulation
📱 Fully responsive layout
🎨 Smooth hover effects & transitions

---

## 🛠️ Tech Stack

* 🧱 HTML5
* 🎨 CSS3 (Flexbox, Grid, Media Queries)
* ⚙️ JavaScript (DOM Manipulation)

## ⚙️ How It Works

1. User enters a prompt
2. System processes input (simulated AI logic)
3. Output appears with typing animation

This mimics the behavior of real AI tools used in modern SaaS products.
